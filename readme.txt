Inbound Marketing & Marketing Automation for WordPress

== Description ==

OpenInbound is like Google Analytics but instead of tracking visits and page views, OpenInbound tracks contacts and interactions on websites.

== Installation ==

1. Sign up on http://openinbound.com/.
1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Settings -> OpenInbound screen to configure the plugin
